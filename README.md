# CyArt VAPT Team 03

## Cybersecurity Practical Labs & VAPT Assessments

This repository contains practical cybersecurity labs, vulnerability assessments, penetration testing exercises, and security research activities performed in authorized lab environments for educational and defensive security purposes.

---

## Included Assessments

### 1. Web Application Security Testing
- Authentication testing
- Session analysis
- Input validation testing
- Burp Suite analysis

### 2. API Security Assessment
- Endpoint enumeration
- Authorization testing
- Token analysis
- API misconfiguration checks

### 3. Privilege Escalation & Persistence
- SUID enumeration
- Cron job persistence
- Linux privilege escalation techniques

### 4. Network Protocol Attacks
- SMB relay simulation
- Responder testing
- ARP spoofing with Ettercap
- Traffic analysis using Wireshark

### 5. Mobile Application Security Testing
- Static APK analysis with MobSF
- Android emulator testing
- Dynamic analysis preparation
- Insecure storage analysis

### 6. Full VAPT Capstone Project
- PTES-based penetration testing workflow
- Vulnerability exploitation
- Remediation planning
- Executive reporting
